📚 Online Bookstore

Full-Stack Web Application — Project Report

ApexPlanet Software Pvt. Ltd. — Internship Capstone Project

1. Project Overview

The Online Bookstore is a full-stack web application that enables users to browse, search, and purchase books online. It features a secure authentication system with email OTP verification, an admin dashboard with Chart.js analytics, full CRUD operations, and a responsive Bootstrap 5 UI.

Project Name: Online Bookstore
Version: 2.0.0
Technology: PHP 8+, MySQL, Bootstrap 5
Duration: Days 37–48
Organization: ApexPlanet Software Pvt. Ltd.
Status: ✅ Complete

6. Security Implementation

  • SQL Injection: All database queries use mysqli prepared statements with parameterized queries
  • XSS Prevention: All user output is escaped with htmlspecialchars()
  • Password Security: Passwords hashed with password_hash(PASSWORD_DEFAULT, 10 cost)
  • Email Verification: 6-digit OTP with 10-minute expiry before account activation
  • Session Security: Session-based authentication with role verification on every admin page
  • Input Validation: Server-side validation on all forms
  • File Upload: Restricted to image types only, renamed on upload
  • .htaccess: Directory listing disabled, sensitive files protected
